# To-Do-List
A basic To-Do List web application built with HTML, CSS, and JavaScript to practice frontend development.

# Project Overview
This project is a **basic To-Do List web application**, created as a **practice project** to improve frontend development skills.  
It is built using **HTML, CSS, and JavaScript** and demonstrates simple task management functionality.

# Features
- Add and delete tasks as completed  
- Responsive and clean user interface  
- Interactive frontend using JavaScript  

# Technologies Used
- HTML5  
- CSS3  
- JavaScript  

# Purpose of This Project
The purpose of this project is to demonstrate:  
- Understanding of HTML page structure  
- CSS styling and layouts  
- JavaScript interactivity for DOM manipulation  
- Basic frontend development skills  

# Folder Structure
- **index.html** — Main HTML page  
- **style.css** — Stylesheet for layout and design  
- **script.js** — JavaScript for adding interactivity  
- **output-screenshots/** — Folder containing screenshots of the project’s output
